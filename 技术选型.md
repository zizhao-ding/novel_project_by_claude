# 技术选型文档

## 项目概述
本项目是一个支持图片和文本小说上传及在线阅读的应用，专注于PC端网页体验。用户可以上传图片和小说文件（TXT格式），并提供流畅的在线阅读体验。

## 前端技术栈

### 核心框架
- **Vue 3**: 最新版本的渐进式JavaScript框架，提供 Composition API 和更好的性能
- **TypeScript**: 为JavaScript添加静态类型检查，提高代码质量和维护性

### 状态管理
- **Pinia**: Vue官方推荐的状态管理库，轻量级且易于使用，具有良好的TypeScript支持

### 路由管理
- **Vue Router**: Vue官方路由管理器，支持嵌套路由、动态路由匹配等功能

### UI 组件库
- **Element Plus**: 基于Vue 3的企业级UI组件库，提供丰富的组件和良好的设计规范

### 构建工具
- **Vite**: 下一代前端构建工具，提供快速的热更新和按需编译
- **ESLint**: 代码质量检测工具
- **Prettier**: 代码格式化工具
- **Husky & lint-staged**: Git钩子工具，用于代码提交前检查

### 其他前端依赖
- **Axios**: HTTP客户端，用于前后端通信
- **vueuse**: Vue组合式函数工具库，提供常用的业务逻辑函数
- **vue-i18n**: 国际化支持
- **crypto-js**: 加密解密库（用于文件哈希等）
- **file-saver**: 文件下载功能
- **marked**: Markdown解析器（如需要富文本展示）

## 后端技术栈

### 核心语言
- **Python 3.9+**: 主要开发语言，具备强大的生态系统和社区支持

### Web框架
- **FastAPI**: 现代、快速（高性能）的Web框架，支持异步编程，自动生成API文档

### 数据库
- **SQLite**: 轻量级嵌入式数据库，无需额外安装和配置，适合中小型应用

### 数据库 ORM/ORM-like 工具
- **SQLModel**: 结合了SQLAlchemy和Pydantic的优点，支持类型提示
- **SQLAlchemy**: Python SQL工具包和对象关系映射器

### 文件处理
- **Pillow**: Python图像处理库，用于图片缩放、格式转换等
- **python-docx**: 处理Word文档（如有需要）
- **pdf2image**: PDF转图片（如有需要）

### 安全相关
- **bcrypt**: 密码哈希库
- **PyJWT**: JSON Web Token实现
- **cryptography**: 提供密码学原语

### 工具和实用程序
- **python-multipart**: 处理multipart/form-data请求（文件上传）
- **python-dotenv**: 环境变量管理
- **uvicorn**: ASGI服务器，用于部署FastAPI应用
- **gunicorn**: 生产环境HTTP服务器（Linux）
- **Pillow**: 图像处理
- **python-slugify**: 生成URL友好的slug

## 数据库设计

### 用户表 (users)
- id (INTEGER, 主键, 自增)
- username (VARCHAR(100), 用户名, 唯一)
- email (VARCHAR(255), 邮箱, 唯一)
- password_hash (VARCHAR(255), 密码哈希)
- created_at (DATETIME, 创建时间)
- updated_at (DATETIME, 更新时间)

### 小说表 (novels)
- id (INTEGER, 主键, 自增)
- user_id (INTEGER, 外键关联用户)
- title (VARCHAR(255), 标题)
- description (TEXT, 描述)
- file_path (VARCHAR(500), 文件存储路径)
- file_size (INTEGER, 文件大小, 单位字节)
- page_count (INTEGER, 页数/段落数)
- upload_date (DATETIME, 上传日期)
- views (INTEGER, 浏览次数, 默认0)
- created_at (DATETIME, 创建时间)
- updated_at (DATETIME, 更新时间)

### 图片表 (images)
- id (INTEGER, 主键, 自增)
- novel_id (INTEGER, 外键关联小说, 可为空)
- user_id (INTEGER, 外键关联用户)
- filename (VARCHAR(255), 原始文件名)
- file_path (VARCHAR(500), 存储路径)
- thumbnail_path (VARCHAR(500), 缩略图路径)
- alt_text (VARCHAR(500), 替代文本)
- upload_date (DATETIME, 上传日期)
- created_at (DATETIME, 创建时间)
- updated_at (DATETIME, 更新时间)

### 书签表 (bookmarks)
- id (INTEGER, 主键, 自增)
- user_id (INTEGER, 外键关联用户)
- novel_id (INTEGER, 外键关联小说)
- position (INTEGER, 阅读位置/页码)
- created_at (DATETIME, 创建时间)

## API 设计规范

### 认证相关接口
- POST /api/auth/register - 用户注册
- POST /api/auth/login - 用户登录
- GET /api/auth/profile - 获取用户资料
- POST /api/auth/logout - 用户登出

### 上传相关接口
- POST /api/upload/novel - 上传小说
- POST /api/upload/image - 上传图片
- GET /api/upload/status/:id - 查询上传状态

### 内容相关接口
- GET /api/novels - 获取小说列表
- GET /api/novels/:id - 获取特定小说信息
- GET /api/novels/:id/content - 获取小说内容
- GET /api/images - 获取图片列表
- GET /api/images/:novelId - 获取特定小说相关图片

### 用户相关接口
- GET /api/user/library - 获取用户图书馆
- POST /api/user/bookmark - 添加/更新书签
- DELETE /api/user/bookmark/:id - 删除书签
- GET /api/user/bookmarks - 获取用户书签列表
